#### ✅ 3. 修复运行测试失败（ModuleNotFoundError）
**错误信息**：
```
ModuleNotFoundError: No module named 'api'
```

**问题原因**：
- pytest执行时PYTHONPATH未包含项目根目录
- 无法导入项目模块（api、utils等）

**解决方案**：
1. 使用虚拟环境的python执行：`.venv/bin/python -m pytest`
2. 设置环境变量：`PYTHONPATH = 项目根目录`
3. 确保pytest能找到所有项目模块

**修改文件**：`pages/2_▶️_运行测试.py`

**代码示例**：
```python
# 使用虚拟环境的python
venv_python = project_root / ".venv" / "bin" / "python"
full_cmd = [str(venv_python), "-m", "pytest"] + cmd_parts[1:]

# 设置PYTHONPATH
env = os.environ.copy()
env["PYTHONPATH"] = str(project_root)

# 执行
subprocess.run(full_cmd, env=env, ...)
```
