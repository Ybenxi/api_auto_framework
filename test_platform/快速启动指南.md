# 测试管理平台 - 快速启动指南

## 🚀 快速启动

### 方法一：使用启动脚本（推荐）

```bash
cd "/Users/mac/api auto/api_auto_framework"
./start_platform.sh
```

### 方法二：直接命令

```bash
cd "/Users/mac/api auto/api_auto_framework"
source .venv/bin/activate  # 激活虚拟环境
streamlit run test_platform/app.py
```

浏览器会自动打开 `http://localhost:8501`

## 📦 首次使用

### 1. 确认streamlit已安装

```bash
pip show streamlit
```

如果未安装，执行：

```bash

```

执行上面的启动命令即可

在终端按 `Ctrl + C`

## 💡 使用技巧

### 1. 运行特定模块测试

```
1. 打开"运行测试"页面
2. 选择"按模块运行"
3. 选择目标模块
4. 点击"开始运行"
```

### 2. 查看测试详情

```
1. 打开"测试用例"页面
2. 选择模块和文件
3. 展开测试场景查看验证点
```

### 3. 管理历史报告

```
1. 打开"历史报告"页面
2. 点击报告链接在浏览器查看
3. 需要时清理旧报告
```

## ⚠️ 常见问题

### Q: 启动后浏览器没有自动打开？
**A**: 手动打开 `http://localhost:8501`

### Q: 提示端口被占用？
**A**: 修改 `test_platform/.streamlit/config.toml` 中的 `port` 配置

### Q: 运行测试卡住不动？
**A**: 检查pytest进程，可能测试确实在运行中（查看日志文件）

### Q: 报告链接点击无反应？
**A**: 复制链接手动粘贴到浏览器地址栏

## 🔧 配置说明

### Streamlit配置文件
位置: `test_platform/.streamlit/config.toml`

可配置：
- 端口号（默认8501）
- 主题颜色
- 浏览器行为

### 项目路径
所有路径都使用相对路径，自动适配项目位置
