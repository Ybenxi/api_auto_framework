# 测试管理平台

基于Streamlit构建的API自动化测试管理平台

## 功能特性

- 📋 **测试用例浏览** - 可视化查看所有测试用例、场景描述、验证点
- ▶️ **一键运行测试** - 支持按模块/文件运行，实时查看输出和统计
- 📊 **历史报告管理** - 浏览、打开、清理历史测试报告

## 使用方法

### 1. 安装依赖

```bash
pip install streamlit
```

或者更新requirements.txt后安装：

```bash
pip install -r requirements.txt
```

### 2. 启动平台

在项目根目录执行：

```bash
streamlit run test_platform/app.py
```

浏览器会自动打开 `http://localhost:8501`

### 3. 使用功能

- **主页**: 查看项目统计信息和模块列表
- **测试用例页**: 浏览测试用例详情和代码
- **运行测试页**: 选择测试运行，查看实时输出
- **历史报告页**: 管理和查看历史测试报告

## 目录结构

```
test_platform/
├── app.py                      # 主入口页面
├── pages/                      # 多页面应用
│   ├── 1_📋_测试用例.py         # 测试用例浏览
│   ├── 2_▶️_运行测试.py         # 运行测试
│   └── 3_📊_历史报告.py         # 历史报告
├── utils/                      # 工具函数
│   └── __init__.py
└── README.md                   # 本文档
```

## 注意事项

1. 确保在项目根目录运行启动命令
2. 运行测试时可能需要几分钟，请耐心等待
3. 报告会自动生成在 `reports/` 目录
4. 清理报告时会保留最新5个和final_report.html
5. 查看大报告（>1MB）时可能需要几秒加载时间
6. 平台会自动使用项目虚拟环境中的pytest

## 技术栈

- **Streamlit**: Web框架
- **Python标准库**: subprocess, pathlib, re
- **项目依赖**: pytest
