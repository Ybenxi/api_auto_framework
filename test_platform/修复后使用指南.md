## 🆕 新功能使用

### 1. 内嵌查看报告

**步骤**：
1. 进入"历史报告"页面
2. 在"👀 查看报告"区域选择报告
3. 点击"📖 在此查看"
4. 报告会在页面内直接显示

**优点**：
- 无需下载
- 即看即走
- 支持滚动查看

**注意**：
- 大报告（>1MB）加载需要几秒
- 可以使用浏览器的Ctrl+F搜索

### 2. 智能模块过滤

**之前**：
- 会显示account_opening等空模块
- 选择后发现没有测试

**现在**：
- 自动过滤空模块
- 只显示有实际测试的模块
- 节省选择时间

### 3. 虚拟环境自动检测

**之前**：
- 可能使用全局pytest
- 版本不匹配导致失败

**现在**：
- 自动使用`.venv/bin/pytest`
- 确保依赖一致
- 提高成功率

## 🐛 如果还有问题

### 问题1：重启后仍显示警告

**解决**：
```bash
# 完全停止
ps aux | grep streamlit | grep -v grep | awk '{print $2}' | xargs kill

# 重新启动
./start_platform.sh
```

### 问题2：报告下载失败

**可能原因**：报告文件过大或损坏

**解决**：
1. 检查`reports/`目录是否存在
2. 尝试其他报告
3. 查看终端错误信息

### 问题3：运行测试仍然失败

**检查**：
```bash
# 验证pytest路径
ls -lh .venv/bin/pytest

# 手动测试
source .venv/bin/activate
pytest test_cases/tenant/ -v
```

### 问题4：界面显示异常

**解决**：
1. 清除浏览器缓存
2. 使用无痕模式重新打开
3. 尝试不同浏览器（Chrome/Edge/Safari）

## 💡 使用技巧

### 技巧1：快速运行单个模块

1. 进入"运行测试"页面
2. 选择"按模块运行"
3. 选择小模块（如tenant，只有6个测试）
4. 点击"开始运行"
5. 约10秒看到结果

### 技巧2：查看最新报告

1. 进入"历史报告"页面
2. 第一个就是🌟最新的`final_report.html`
3. 直接点击"📖 在此查看"
4. 或下载到本地慢慢看

### 技巧3：清理旧报告释放空间

1. 进入"历史报告"页面
2. 查看报告总大小
3. 如果超过5个，点击"🧹 清理旧报告"
4. 只保留最新5个
