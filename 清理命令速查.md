## 🔄 恢复数据（SQL）

```sql
-- 查看备份表
SELECT table_name FROM information_schema.tables 
WHERE table_schema = 'backup_data' 
  AND table_name LIKE 'actc_%'
ORDER BY table_name DESC;

-- 恢复示例（根据实际表名调整）
INSERT INTO actc.contact
SELECT * FROM backup_data.actc_contact_0209;

INSERT INTO actc.t_share_sub_account
SELECT * FROM backup_data.actc_t_share_sub_account_0209;

INSERT INTO actc.t_share_fbo_account
SELECT * FROM backup_data.actc_t_share_fbo_account_0209;
```
