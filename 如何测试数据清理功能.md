## 🐛 故障排查

### 问题1: 连接被拒绝

**错误**：
```
Connection refused
Is the server running on that host and accepting TCP/IP connections?
```

**原因**：数据库未配置或配置错误

**解决**：检查 `.env` 文件是否存在且配置正确

---

### 问题2: 密码验证失败

**错误**：
```
password authentication failed for user "developer"
```

**原因**：数据库凭据不正确

**解决**：请确认正确的数据库用户名和密码

---

### 问题3: IP不在白名单

**错误**：
```
no pg_hba.conf entry for host "X.X.X.X"
```

**原因**：数据库有IP白名单限制

**解决**：
1. 在本地运行（本地IP可能已在白名单）
2. 或联系DBA将你的IP加入白名单
